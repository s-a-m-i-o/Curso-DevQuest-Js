Operadores Lógicos

Servem papa validar dados ou verrificar condições

==

IGUAL

a==b - Verdadeiro se A for igual a B

Exemplo

const a = 3

const b = 3

cosnsole.log(a==b)

===

IDENTICO

a===b - Verdadeiro se A for identico a B

Exemplo

const a = 3

const b = 3

cosnsole.log(a===b)

!==

NÃO IDENTICO

Compara se os dois valores não são identicos

a==b - Verdadeiro se A não for identico a B

Exemplo

const a = 3

const b = 3

cosnsole.log(a!==b)

!=

DIFERENTE

a==b - Verdadeiro se o A for diferente de B

Exemplo

const a = 3

const b = 3

cosnsole.log(a!=b)

<

MENOR

a < b - Verdadeiro quando o A for menor que o B

Exemplo

const a = 2

const b = 3

cosnsole.log(a < b)

<=

MENOR E/OU IGUAL

a < b - Verdadeiro quando o A for menor ou igual ao B

Exemplo

const a = 2

const b = 3

cosnsole.log(a <= b)

>

MAIOR

a > b - Verdadeiro quando o A for maior que o B

Exemplo

const a = 2

const b = 3

cosnsole.log(a > b)
